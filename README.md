# AI VoiceOver Data – The Burning Crusade

AI-generated voiceover data pack for **World of Warcraft: The Burning Crusade** (Classic Era / 2.4.3), compatible with the [AI VoiceOver](https://github.com/mrthinger/wow-voiceover) addon.

> **Note:** This is a community data module — it provides the actual audio files and lookup tables. You need the **AI_VoiceOver** addon installed for the voices to play in-game.

---

## Contents

All audio is generated using **Qwen3-TTS** (open-source local TTS) with voice-cloned samples per race and gender.

---

## Installation

1. Make sure **AI_VoiceOver** is installed in `World of Warcraft\Interface\AddOns\`
2. Copy the **AI_VoiceOverData_TBC** folder into the Interface folder
3. Profit!

---

## Supported Quests

Covers (hoipefully) all quests across all TBC zones:

- Hellfire Peninsula
- Zangarmarsh
- Terokkar Forest
- Nagrand
- Blade's Edge Mountains
- Netherstorm
- Shadowmoon Valley
- Isle of Quel'Danas
- All dungeon attunement quest chains
- Class quests (Warlock mount, Paladin mount, etc.)
- Major dungeon & raid quests (Hellfire Citadel, Coilfang Reservoir, Tempest Keep, Black Temple, Sunwell Plateau)

---

## Voice Coverage

NPCs without a matching voice reference will mostly be voiced by the narrator (definetly not Morgan Freeman). If you find any quest that dont play audio, please open an issue! Thanks!
